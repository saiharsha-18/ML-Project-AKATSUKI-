2. Decision Tree
3. Random Forest
4. Gradient Boosting

# DATASET
The dataset for this project is downloaded from here:
https://www.kaggle.com/kaushil268/disease-prediction-using-machine-learning

This dataset has 3 columns:
Disease  | Count of Disease Occurrence | Symptom
